> 시도 해보면서 느낌 보기

- Outside-In 개발과 API 우선 설계 접근 방식
- '테스트 주도 개발로 배우는 객체 지향 설계와 실천'에서 학습한 내용 적용
- 연관관계 없이 개발
- Mock 대신 Stub 사용한 테스트 코드 작성
